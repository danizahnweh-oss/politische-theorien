# Politische Theorien

Eigenständige Unterrichtswebsite für Politik und Gesellschaft, Q12 Leistungsfach am Gymnasium in Bayern.

## Inhalt

- Wortschatztraining
- aktueller Aufhänger zur Frage "Wie viel Staat braucht Freiheit?"
- zwei 90-Minuten-Unterrichtsblöcke
- Arbeitsphasen zu Locke, Marx, Burke, Montesquieu und Rousseau
- dauerhaft erreichbare Operatoren-Hilfe
- Endprodukt: selbst erstelltes Schaubild

## Lokal starten

```bash
python3 -m http.server 8010
```

Dann im Browser öffnen:

```text
http://127.0.0.1:8010/
```
